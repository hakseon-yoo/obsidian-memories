# Module 2 · 스케줄링

> 상위 문서: [[00 - AX-2 쉬운 설명서 (Index)]]

> [!summary] 한 줄로 말하면
> 영업이 확정한 수주를 받아 **생산관리팀이 외주처와 일정을 협의·확정**하고, 영업·고객사·생산팀에 통보하는 모듈.

> [!info] 책임 분리
> 매일 작업 지시 + 실행 중 납기/품질 이슈 처리는 **[[04 - Module 3 · 생산 관제]]** 로 분리되었음 (PPT (13)(14) 영역).
> M2 는 **일정 짜기·확정·재조정** 책임 — PPT 의 "스케줄링 모듈" 영역만.

- **관련 주체**: 생산관리팀 ↔ 외주처(다수) · 영업팀 (납기 조정 협의)
- **PPT 워크플로우 매핑**: (4) 수주 내역 받음 · (5) 생산 계획 전달 · (6)/(7) 외주처 컨펌·조정 핑퐁 · (8) 확정 · (9) 영업·고객 통보 · (12) 생산팀 전달 · (17) 스케줄 재조정
- **핵심 요소**: 외주 발주 자동화 / 외주처 컨펌 핑퐁 / 납기 협의 흐름 (요청↔조정↔확정 3컬럼) / 재조정 트리거

---

## 실제 현장 장면 (Before)

- 생산관리팀이 엑셀에 SKU·수량·납기일을 적어 카톡으로 외주처에 배포.
- 외주처 회신을 일일이 받아 ERP에 수기 입력 — 거의 하루 소요 (PPT Slide 2: "외주처 진도 관리 반나절 이상").
- 외주처가 답을 안 주면 생산관리가 카톡으로 계속 독촉 → 회신 확보 어려움.
- 납기 못 맞춘 건은 영업이 고객사와 별도 통화 후 조정 납기일을 ERP 에 다시 기입.

> ※ "납기 임박 컨펌 / 품질 판정 요청" 의 매일 진행 추적은 **[[04 - Module 3 · 생산 관제]]** 영역.

## AX-2 적용 후 (After)

- 수주 확정 즉시 시스템이 SKU → 공정 순서·업체 자동 파악 (ERP 매핑 활용).
- **각 외주처에 시점·완료기한 자동 통지** (PPT (5)) — 예: "1번 업체: 일주일 후 완료 가능?", "2번 업체: 일주일 후 물량 들어오는데 그 후 3일 가능?"
- 외주처가 시스템에서 컨펌 / 조정 회신 (PPT (6)/(7)) → ERP 에 확정계획 자동 반영 (PPT (8)).
- 회신이 없으면 시스템이 **자동 너징(독촉)** ("미기입 시 지속 요청") — AX-2 핵심 가치.
- 영업·고객사 통보 (PPT (9)) — 납기 조정 시 `요청 / 조정 / 확정` 3컬럼으로 추적.
- 외주처 변경 시 ERP / SKU 마스터 메타 자동 동기화.

> [!info] 초기 운영 보완책 (2026-05-07 합의)
> 시스템 초창기에는 **생산관리팀이 자동 산정 결과를 한번 검토** 후 외주처에 발송 → 신뢰도 쌓이면 직접 발송. 자동화 점진 적용.

> [!info] 외주처 화면 (2026-05-07 합의)
> 외주처 화면에는 **신규 건만 보는 게 아니라 기존 기수주 건까지 함께 조회** 가능. 같은 업체에 여러 제품 누적 시 전체 맥락 보고 응답.

> [!warning] 미정 (후속 결정)
> **배치 전송 vs 건별 전송** — 수주 확정 때마다 한 건씩 vs 하루 1~2회 묶음. 후속 미팅에서 결정.

## 개발팀이 구현해야 할 기능

- [ ] (5)/(6)/(7)/(8) 생산 계획 전달-외주처 컨펌·조정 핑퐁-확정의 ERP 자동 입력 파이프라인
- [ ] (9) 영업 → 고객 협의 → **조정 납기일 기입** 흐름 (Order entity 의 `요청/조정/확정` 3컬럼)
- [ ] (12) 확정 계획을 M3 (생산 관제) 로 전달
- [ ] (17) 스케줄 재조정 — M3 (14) 납기 조정·품질 판정 요청 발생 시 (5)~(16) 반복 트리거
- [ ] 외주처 변경 시 ERP/SKU 마스터 자동 동기화 (PPT Slide 2 메모)
- [ ] BOM 펼치기 → 부자재 필요량 산출 → 부속 영역(구매·정산)에 신호

---

> [!warning] 개발 시 반드시 체크
> - 외주처 **컨펌 핑퐁 흐름**의 노티 채널·빈도 미확정 (오늘 미팅 Q2)
> - **납기 협의** (요청/조정/확정) Order entity 컬럼 분리 설계 필요 (PPT (9))
> - **외주처 변경 권한** — 외주관리팀 신설 시 권한 이관 (PPT Slide 2 메모)
> - **24시간 이내 ERP 자동 발주** — 첫 날짜 조정 로직 미확정 (PPT Slide 2 메모)
> - 매일 작업 지시 / 임박 컨펌 / 품질 판정은 **[[04 - Module 3 · 생산 관제]]** 책임

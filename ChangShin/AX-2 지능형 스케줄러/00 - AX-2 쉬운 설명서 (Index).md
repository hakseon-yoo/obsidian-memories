## 비유로 이해하기

> [!tip] 한 장 요약
> **"학교 시간표 + 택배 배송 현황판 + 가계부"**가 하나로 합쳐진 시스템.
>
> 주문(수주)이 들어오면 → 언제까지 만들 수 있는지 자동으로 알려주고 → 공장/외주처에 일감을 뿌리고 → 매일 지켜지는지 지켜보고 → 배송 트럭까지 잡아주고 → 돈 정산도 해준다.

| 모듈 | 이름 | 역할 |
|------|------|------|
| **M1** | [[02 - Module 1 · 납기일 예측\|납기일 예측]] | 주문 들어오면 언제까지 만들 수 있는지 자동 계산 |
| **M2** | [[03 - Module 2 · 생산 계획 관리\|생산 계획 관리]] | 외주처에 일감 뿌리고 매일 진행 상황 추적 |
| **M3** | [[04 - Module 3 · 물류 · 배차\|물류 · 배차]] | 트럭 · 창고 · 입고 예약까지 자동 슬롯 생성 |
| **M4** | [[05 - Module 4 · 구매 · 정산\|구매 · 정산]] | BOM · 공제 · 거래명세표까지 자동 정산 |

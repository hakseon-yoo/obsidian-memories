# 사전 준비 · 등장인물 · 용어집

## 함께 일하는 팀 (등장인물)

| 팀/주체 | 역할 |
|--------|------|
| **영업팀** | 주문을 따와서 시스템에 입력 |
| **생산관리팀** | "컨트롤 타워" · 일정 짜고 외주에 뿌리고 확인 |
| **외주처** | 실제로 물건을 만드는 협력 업체 (A, B, C…) |
| **품질팀** | 불량 여부 판정 · Go/No Go 결정 |
| **자재팀** | 물류 · 배차 · 창고 예약 담당 |
| **구매팀** | 돈 정산 · 발주 · 공제 |
| **금형관리팀** | 금형 상태와 메인터넌스 관리 |
| **ODM / 고객사** | 완제품을 납품받는 최종 업체 |

## 꼭 알아야 할 용어

> [!info] 용어집
> - **수주 (受注)**: "주문 받기" — 고객사가 물건 만들어달라고 주문한 것
> - **SKU**: 상품 종류 하나하나의 고유 ID (예: SKU A = ○○모델 빨강)
> - **BOM**: Bill of Materials — 완제품 1개 만들 때 필요한 부품 · 재료 리스트
> - **WIP**: Work In Progress — 만드는 중인 반제품 (완성되지 않은 상태)
> - **납기일**: "언제까지 줄게요" 약속한 날짜
> - **외주처**: 일부 공정을 대신 만들어 주는 협력 업체
> - **T0 / T1 / T2**: 신제품 개발 단계 (0차 · 1차 · 2차 시제품)
> - **공제**: 정산할 때 제품 불량/재생산 이유로 금액을 깎는 것
> - **ERP**: 회사 업무 관리 시스템 (주문 · 재고 · 발주 등)
> - **PDA**: 손에 들고 다니는 바코드/QR 스캐너

# Module 3 · 물류 · 배차 관리

> 상위 문서: [[00 - AX-2 쉬운 설명서 (Index)]]

> [!summary] 한 줄로 말하면
> 확정된 생산 일정 기반으로 **물류 슬롯을 자동 생성**하고, 물류 유형별 필수 정보(거래명세표 · QR · 창고예약)를 빠짐없이 기입하도록 돕는 모듈.

- **관련 주체**: 자재팀 중심 · 외주처/ODM 연결
- **핵심 요소**: 5종 물류 유형 자동 구분 / ODM 창고 예약 노티 / 거래명세표 · QR 첨부

---

## 5가지 물류 유형별 필수 정보 · 노티

| 물류 유형 | 필수 정보 | 노티(알림) 기능 |
|----------|----------|----------------|
| **창신 → 외주처** (반제품) | PDA 기반 전표 데이터, QR, 배차 여부 (트럭번호 · 운반자 연락처) | 창신에서 반제품 생산 후 **3일간 출고 없으면** 외주업체에 배차 필요 여부 확인 노티 · 확정 시 자재팀 노티 |
| **외주처 → 외주처** (반제품) | 거래명세서 | — |
| **외주처 → 창신** (반제품/완제품) | 거래명세서 · 창신 검수 완료 여부 · 반품 여부 | 품질/수량 부족 등으로 반품 시 생산관리팀에 노티 → 생산 일정 즉시 재조정 |
| **외주처 → 고객사** (완제품) | 거래명세서 · QR(ODM/브랜드사 입고용) · ODM 창고 예약 내역 | ODM 예약 전 **D-7/3/1 노티**, 예약 후 정보 미기입 시 자재팀 별도 노티, **납기 전일 15시까지** 거래명세서 미등록 시 노티 |
| **창신 → 고객사** (완제품) | 거래명세서 · QR(ODM/브랜드사 입고용) · ODM 창고 예약 내역 | ODM 예약 전 **D-7/3/1 노티**, 예약 후 정보 미기입 시 자재팀 별도 노티 |

---

> [!warning] 개발 시 체크
> 외주처 간 이동은 대부분 자체 트럭/소량 → 슬롯 생성 범위는 **"창신 In/Out + 완제품 납품"**으로 제한 검토.
> 외주처 PDA 활용 가능 여부 · 창신 입고 시 불량/수량 부족 ERP 미입력 이슈 확인 필요.

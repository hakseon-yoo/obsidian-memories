# Main Vault — Claude 글로벌 지침

이 vault는 **Claude의 글로벌 영구 메모리**다. 모든 프로젝트가 공유하는 지식·룰·결정을 담는다.

## 진입 규칙 (필수)

1. 세션 시작 시 가장 먼저 `_index.md`를 읽는다.
2. 그 다음 작업과 관련된 **카테고리 인덱스**(`_xxx_index.md`)만 읽는다.
3. 카테고리 인덱스에서 필요한 개별 노트로만 진입한다.
4. **vault 전체를 스캔하지 않는다.**

## 노트 작성 규칙

- 모든 노트는 [[00_System/schema|Frontmatter Schema]]를 준수한다.
- 새 노트는 반드시 `_templates/`의 템플릿을 경유해 생성한다.
- 노트는 한 개념 = 한 노트(원자성). 합치거나 길게 늘이지 않는다.
- 영구적으로 유효한 지식만 이 vault에 둔다. 휘발성 작업 노트는 프로젝트 볼트로.

## 분류 가이드

| 종류 | 위치 |
|---|---|
| 평생 유효한 개념·지식 | `10_Knowledge/` |
| 외부 자료의 본인 요약 | `20_Resources/` |
| 도구·스타일 등 평생급 결정 | `30_Decisions/` |
| 프로젝트 메타데이터 | `40_Projects/_projects_registry.md` |
| 사용자 코딩 스타일·운영 룰 | `00_System/rules.md` |

## 금지

- vault 전체 스캔
- frontmatter 누락 노트 생성
- `90_Archive/` 수정
- 프로젝트 한정 정보를 이 vault에 작성 (→ 프로젝트 볼트로)

## 프로젝트 볼트와의 관계

- 각 프로젝트는 자체 `vault/` 폴더를 가진다.
- 프로젝트 볼트의 `CLAUDE.md`는 항상 이 메인 볼트를 먼저 참조하라고 명시한다.
- 프로젝트 작업 중 영구 지식이 발견되면 **이 vault로 승격**한다.
